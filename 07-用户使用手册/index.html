    用户使用手册¶

    本文档介绍 BeiDou 系统的玩家使用指南和管理员后台操作手册。

    目录¶

    1. 玩家使用指南
    2. 管理员操作手册
    3. 常见问题解答

    玩家使用指南¶

    客户端安装步骤¶

    1. 下载客户端¶

    从官方 Release 下载完整的客户端和服务端打包文件：

    https://github.com/BeiDouMS/BeiDou-Server/releases

    2. 安装 ijl15.dll¶

    步骤：

    1. 进入游戏客户端目录
    2. 找到原有的 ijl15.dll 文件
    3. 将其重命名为 2ijl15.dll（作为备份）
    4. 将新编译的 ijl15.dll 复制到客户端目录

    文件来源：

    • 自行编译：BeiDou-ijl15/out/Release/ijl15.dll
    • Release下载：解压后获取

    3. 复制配置文件¶

    将 config.ini 文件复制到游戏客户端目录。

    配置文件位置：

    • 自行编译：BeiDou-ijl15/ezorsia/config.ini
    • Release下载：解压后获取

    4. 配置服务器地址¶

    编辑 config.ini，设置服务器连接信息：

    [general]
;服务器地址
ServerIP_Address=127.0.0.1    # 修改为服务器IP
serverIP_Port=8484             # 游戏端口，默认8484

    5. 配置分辨率¶

    根据显示器调整分辨率：

    [general]
width=1280    # 分辨率宽度
height=720    # 分辨率高度

    推荐分辨率： - 1280x720 - 1366x768 - 1600x900 - 1920x1080

    config.ini 配置指南¶

    中文输入配置¶

    [general]
;IME 修复方案
imeType=1           # 优先尝试方案1

;加载文字汉化
SwitchChinese=true  # 开启汉化

    如果方案1导致"卡门"问题，请改为 imeType=0。

    窗口模式配置¶

    [general]
WindowedMode=true   # 窗口模式
RemoveLogos=true    # 跳过开场动画

    属性上限突破¶

    [optional]
;突破原版属性上限
setDamageCap=199999     # 物理攻击上限
setMAtkCap=1999         # 魔攻上限
setAccCap=999           # 命中上限
setAvdCap=999           # 回避上限
speedMovementCap=140    # 移速上限
jumpCap=123             # 跳跃上限

    游戏功能说明¶

    新增功能列表¶

    BeiDou 相比原版新增以下功能：

    功能 说明
    中文输入 支持中文输入，解决卡门问题
    中文角色名 支持创建中文角色名
    滚轮修复 修复鼠标滚轮乱飞问题
    ToolTip修复 修复提示框超出窗口问题
    长快捷键 支持长键盘快捷键
    交易中心居中 交易中心窗口居中显示
    中文汉化 游戏界面中文汉化
    属性上限突破 魔攻/魔防/命中/回避/跳跃上限突破
    Boss血量显示 Boss血条头像下方显示百分比
    免密模式 配合服务端实现免密登录

    中文输入设置¶

    BeiDou 完整支持中文输入：

    • 角色名支持中文
    • 聊天框支持中文
    • 搜索框支持中文

    注意事项：

    如果遇到中文输入卡顿或卡门：

    1. 将 imeType 改为 0
    2. 重启游戏客户端

    装备汉化相关¶

    针对中文环境的调整：

    1. 装备tooltip字体大小优化
    2. 道具有效期字体大小调整
    3. 有效期日期顺序修复
    4. 聊天栏选项汉化（远征队）
    5. 解决 Eqp、Etc 汉化后崩溃问题
    6. 解决 Use 汉化后吃药没声音问题

    游戏启动¶

    启动步骤¶

    1. 确认 ijl15.dll 和 config.ini 在客户端目录
    2. 确认 2ijl15.dll 备份文件存在
    3. 双击启动游戏
    4. 输入账号密码登录

    首次登录¶

    如果是首次登录且服务器开启自动注册：

    • 直接输入新账号名和密码
    • 系统自动创建账号

    管理员操作手册¶

    Web后台访问¶

    访问地址¶

    http://localhost:8787

    或生产环境：

    http://your-server:8787

    登录账号¶

    默认管理员账号：

    账号 密码 角色
    admin admin 管理员

    安全提醒： 生产环境必须修改默认密码。

    用户管理¶

    查询用户¶

    1. 进入"用户管理"页面
    2. 可搜索用户名或ID
    3. 查看用户列表

    用户详情¶

    点击用户可查看：

    • 账户信息
    • 角色列表
    • 登录记录
    • 封禁状态

    封禁用户¶

    1. 选择目标用户
    2. 点击"封禁"按钮
    3. 输入封禁原因
    4. 设置封禁时长（可选）
    5. 确认封禁

    封禁类型：

    类型 说明
    临时封禁 设置解封时间
    永久封禁 不设置解封时间

    解封用户¶

    1. 选择已封禁用户
    2. 点击"解封"按钮
    3. 确认解封

    角色管理¶

    查询角色¶

    1. 进入"角色管理"页面
    2. 可搜索角色名或ID
    3. 查看角色列表

    角色详情¶

    点击角色可查看：

    • 基本信息（等级、职业、金币）
    • 属性详情（STR、DEX、INT、LUK）
    • 装备列表
    • 背包物品

    修改角色¶

    可修改角色属性：

    • 调整等级
    • 调整金币
    • 调整属性值
    • 添加/删除物品

    删除角色¶

    1. 选择目标角色
    2. 点击"删除"按钮
    3. 确认删除

    注意： 删除操作不可恢复，请谨慎操作。

    服务器监控¶

    查看服务器状态¶

    进入"服务器状态"页面可查看：

    信息 说明
    运行状态 是否正常运行
    在线人数 当前在线玩家数
    频道状态 各频道状态
    系统负载 CPU、内存使用率
    运行时间 服务运行时长

    在线玩家列表¶

    可查看在线玩家：

    • 角色名
    • 等级
    • 所在地图
    • 在线时长

    配置修改¶

    游戏参数配置¶

    进入"游戏配置"页面可修改：

    大区配置：

    参数 说明
    经验倍率 exp_rate
    金币倍率 meso_rate
    掉落倍率 drop_rate
    频道数 channel_size
    服务器信息 server_message

    服务器配置：

    参数 说明
    频道容量 channel_capacity
    怪物刷新间隔 respawn_interval
    物品消失时间 item_expire_time

    配置修改流程¶

    1. 找到目标配置项
    2. 修改参数值
    3. 点击保存
    4. 配置自动生效（部分需重启）

    商店管理¶

    查看商店¶

    可查看服务器商店：

    • 商店ID
    • NPC ID
    • 商店物品列表

    修改商店物品¶

    可调整：

    • 物品价格
    • 物品库存
    • 物品顺序

    常见问题解答¶

    玩家常见问题¶

    Q: 登录后崩溃怎么办？¶

    A: 检查以下问题： 1. 确保 DLL 文件正确替换 2. 尝试修改 imeType=0 3. 检查分辨率设置是否兼容 4. 确认服务器地址和端口正确

    Q: 中文输入卡门怎么办？¶

    A: 将 config.ini 中的 imeType 改为 0。

    Q: 滚轮乱飞怎么解决？¶

    A: BeiDou 已修复此问题，确保使用正确的 ijl15.dll。

    Q: 如何调整分辨率？¶

    A: 编辑 config.ini：

    width=1920
height=1080

    Q: Boss血量显示在哪里？¶

    A: 在 Boss 血条的头像下方显示百分比。

    Q: 如何使用免密登录？¶

    A: 需要服务端配合开启免密模式： - 服务端：game_config 表设置 no_password=true - 客户端：config.ini 设置 debug=true 和 noPassword=true

    管理员常见问题¶

    Q: 后台无法登录？¶

    A: 检查： 1. API服务是否运行（端口8686） 2. 前端是否运行（端口8787） 3. 账号密码是否正确 4. JWT配置是否正确

    Q: 配置修改不生效？¶

    A: 部分配置需要重启服务端： - 大区配置（倍率等）：通常即时生效 - 核心配置：可能需要重启 - 网络配置：需要重启

    Q: 数据库表没有创建？¶

    A: 确保： 1. MySQL服务已启动 2. 数据库连接配置正确 3. Flyway自动创建表

    Q: 如何添加新用户？¶

    A: 两种方式： 1. 开启自动注册，用户首次登录自动创建 2. Web后台手动创建

    Q: 如何修改管理员密码？¶

    A: 1. 数据库修改：UPDATE accounts SET password = 'new_password' WHERE name = 'admin' 2. Web后台修改：用户管理页面
